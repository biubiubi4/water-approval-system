# Python AI服务安装指南

## 快速安装

### 方案一：使用虚拟环境（推荐）

Windows PowerShell:
```powershell
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
.\venv\Scripts\Activate.ps1

# 升级pip
python -m pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

Windows CMD:
```cmd
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
venv\Scripts\activate.bat

# 升级pip
python -m pip install --upgrade pip

# 安装依赖
pip install -r requirements.txt
```

### 方案二：直接安装到全局

```bash
pip install --upgrade pip
pip install fastapi uvicorn python-multipart langchain langchain-core langchain-community langchain-text-splitters chromadb pypdf python-docx pydantic-settings sentence-transformers pydantic
```

## 验证安装

创建一个简单的测试文件 `test_install.py`:

```python
from fastapi import FastAPI
from fastapi import File, UploadFile
import uvicorn

app = FastAPI()

@app.get("/")
def hello():
    return {"message": "安装成功！"}

@app.post("/upload")
async def upload(file: UploadFile = File(...)):
    return {"filename": file.filename}

if __name__ == "__main__":
    print("启动测试服务器...")
    uvicorn.run(app, host="0.0.0.0", port=8000)
```

运行测试:
```bash
python test_install.py
```

访问 http://localhost:8000 看到"安装成功！"说明依赖安装正确。

## 常见问题

### 问题1：pip 安装慢
使用国内镜像源：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 问题2：缺少 python-multipart
单独安装：
```bash
pip install python-multipart
```

### 问题3：版本冲突
先卸载后重新安装：
```bash
pip uninstall fastapi python-multipart
pip install fastapi python-multipart
```

## 启动AI服务

安装完成后，运行主程序：
```bash
python main.py
```

访问 http://localhost:8000/docs 查看API文档。
