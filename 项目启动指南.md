# 取水申请材料合规性检测系统 - 快速启动指南

## 项目概述
本项目专为**取水许可申请材料的智能合规性检测**而设计，通过AI技术自动审核申请材料的完整性和合规性。

## 系统架构

采用双栈架构：

- **Java后端** (Spring Boot): 主业务系统，端口8080
- **Python AI服务** (FastAPI + LangChain): 智能审核引擎，端口8000
- **Vue前端**: 用户界面，端口3000

## 环境要求

- Java 17+
- Python 3.9+
- Node.js 18+
- Maven 3.8+

## 快速启动

### 第一步：启动Python AI服务

打开终端1（进入python-ai-service目录）：

```bash
cd python-ai-service

# 使用虚拟环境（推荐）
python -m venv venv

# Windows激活虚拟环境
venv\Scripts\activate

# 安装依赖（使用清华镜像加速）
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

# 启动服务
python main.py
```

访问 http://localhost:8000/docs 查看API文档

### 第二步：启动Java后端

打开终端2（进入java-backend目录）：

```bash
cd java-backend

# 编译并启动
mvn spring-boot:run
```

访问 http://localhost:8080 或 http://localhost:8080/h2-console 查看H2数据库

### 第三步：启动前端

打开终端3（进入frontend目录）：

```bash
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000

## 核心功能说明

### 1. MCP协议工具

已实现两个核心工具，专门用于取水申请审核：

- **knowledge_search**: 从知识库中搜索取水相关的法规文档（水法、取水许可条例等）
- **check_completeness**: 检查取水申请材料的完整性（必填字段、附件等）

API端点：
- `GET /api/mcp/tools` - 获取工具列表
- `POST /api/mcp/{tool_name}` - 调用MCP工具

### 2. LangChain Agent智能审核

完整的取水申请审核Agent实现：

- **形式审查**: 检查申请材料完整性
- **内容规范**: 验证信息填写正确性
- **实质合规**: 结合知识库进行合规判断
- **智能建议**: 生成修改意见和法规依据

### 3. 知识库系统

- 使用ChromaDB向量数据库
- 内置取水相关法规（水法、取水许可条例、河道管理条例等）
- 支持PDF、Word、TXT文档上传和解析
- 支持Embedding向量化存储

### 4. 数据库

Java后端使用H2内存数据库，无需额外配置，开箱即用。

## 常见问题

### Q: Python依赖安装太慢？
A: 使用清华镜像源：`pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple`

### Q: 提示缺少python-multipart？
A: 单独安装：`pip install python-multipart`

### Q: H2数据库如何访问？
A: 访问 http://localhost:8080/h2-console，JDBC URL填 `jdbc:h2:mem:waterapproval`

### Q: 没有真实LLM也能运行吗？
A: 可以！当前Agent实现了不依赖LLM的规则审核逻辑，同时预留了LangChain接口供后续集成。

### Q: 如何添加新的取水法规到知识库？
A: 可以通过 `/api/knowledge/upload` 上传PDF/Word文档，或 `/api/knowledge/add` 添加文本。
